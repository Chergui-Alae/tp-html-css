# TP HTML/CSS

## Description
Projet de base pour les futurs TP PHP.  
Il contient :
- Une page d'accueil
- Un formulaire complet (prêt pour PHP)
- Une page de confirmation statique

## Structure
- `index.html` : page d'accueil
- `pages/formulaire.html` : formulaire
- `pages/confirmation.html` : page de résultat
- `assets/css/style.css` : feuille de style unique

## Choix réalisés
- **HTML sémantique** : header, main, footer, section, fieldset, etc.
- **CSS pur** (sans framework) avec Flexbox pour la navigation
- **Palette simple** : bleu/gris (#2c3e50, #3498db)
- **Formulaire** : tous les champs demandés, labels associés, attributs `name` et `required`

## Exécution
Ouvrir `index.html` dans un navigateur (pas de serveur nécessaire).
