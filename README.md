# TP HTML/CSS

## Description
Projet de base pour les futurs TP PHP.  
Il contient :
- Une page d'accueil
- Un formulaire complet (prêt pour PHP)
- Une page de confirmation statique

## Structure
- `index.html` : page d'accueil
- `pages/formulaire.html` : formulaire
- `pages/confirmation.html` : page de résultat
- `assets/css/style.css` : feuille de style unique

## Exécution
Ouvrir `index.html` dans un navigateur (pas de serveur nécessaire).
